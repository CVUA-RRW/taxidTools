taxidTools: A Python Toolkit for Taxonomy¶

Release v. (Installation)

** taxidTools** is a Python library to handle Taxonomy definitions.

Usage example:

>>> some code example**

taxidTools provides classes and methods to easily handle complex taxonomies from different sources and perform common operations such as consensus determination, finding a last common ancestor or calculating node distances.
