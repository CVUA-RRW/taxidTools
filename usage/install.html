Installation guide¶

This part presents the different possibilities to install taxidTools. To run properly make sure you are using Python 3.9 or above.

Github¶

To get the latest development version, clone the git repository:

$ git clone https://github.com/CVUA-RRW/taxidTools

Once you have a copy of the repository, you can transfer it to your own Python package or install it using pip:

$ cd taxidTools
$ python -m pip install .
